# Screenshot Instructions

This directory should contain the following screenshots for the Mosaic Jekyll theme:

## Required Screenshots

### Desktop Views (1920x1080 or similar)
1. **desktop-home.png** - Homepage showing the blog post grid/list
2. **desktop-post.png** - Individual blog post page showing typography and layout

### Mobile Views (375x812 or iPhone size)
3. **mobile-home.png** - Homepage on mobile device
4. **mobile-post.png** - Blog post on mobile device

### Feature Screenshots
5. **navigation.png** - Navigation menu (both desktop and mobile hamburger)
6. **blog-cards.png** - Close-up of blog post cards showing metadata

## How to Take Screenshots

### Option 1: Using Jekyll Serve
1. Run `bundle exec jekyll serve`
2. Open http://localhost:4000 in your browser
3. Use browser developer tools to capture different viewport sizes

### Option 2: Using Browser DevTools
1. Open Chrome/Firefox DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Select device presets or custom sizes
4. Use the screenshot feature in DevTools

### Option 3: Using Screenshot Tools
- **Mac**: Cmd+Shift+4 for selection, Cmd+Shift+5 for options
- **Windows**: Win+Shift+S for snipping tool
- **Linux**: Use Flameshot or similar tools

### Recommended Sizes
- Desktop: 1920x1080 or 1440x900
- Mobile: 375x812 (iPhone X/11/12) or 414x896 (iPhone 11 Pro Max)
- Feature shots: As needed, but keep consistent aspect ratios

### Tips for Good Screenshots
1. Clear browser cache before taking screenshots
2. Use incognito/private mode to avoid extensions
3. Ensure good contrast and readability
4. Include some actual content (not just Lorem Ipsum)
5. Show the theme's key features prominently

## File Format
- Use PNG format for all screenshots
- Optimize file size (use tools like TinyPNG if needed)
- Keep file sizes under 500KB each for faster loading
